Evolução das temporadas de animação japonesa

  • GitHub
  • Medium

Motivação

O site anime-planet possui notas e recomendações dos usuários para animações japonesas, possuindo um acervo com mais de 17000 animações datando do século passado até hoje.

Como a TV japonesa costuma ser dividida em quatro temporadas no ano, resolvi fazer um web scrapping para investigar essa sequência histórica e praticar minhas habilidades com análise de séries temporais.

Perguntas

  • As temporadas tem ficado melhores?
  • Qual é a melhor temporada do ano?
  • Qual a previsão para o próximo ano?

Dados

O banco consiste apenas de animações televisivas. Filmes, animações originais, especiais e outras categorias ficaram de fora do web scrapping.

Além disso, possui no máximo os 35 melhores por temporada. Totalizando 2633 linhas e 4 colunas como exemplo abaixo:

Name AverageRating Season Year
Slayers 4.1 spring 1995
Tenchi Universe 4.0 spring 1995
... ... ... ...
Chou Futsuu Toshi Kashiwa Densetsu R 1.7 summer 2020
Ninja Collection 1.4 summer 2020

Técnicas

Foram testadas diferentes técnicas para modelagem de séries temporais. Os quatro modelos que mostraram o melhor desempenho foram:

  • SARIMA
  • ETS
  • TBATS
  • NNAR

Resultados

A série apresenta tendência, heterocedasticidade e sazonalidade e a combinação dos quatro modelos apresentou melhor acurácia sobre o conjunto de dados teste. Então, as temporadas tem se tornado melhores e a primavera tende a ser a melhor temporada. Porém, nos anos recentes a média das notas das temporadas tem sido mais homogênea.
