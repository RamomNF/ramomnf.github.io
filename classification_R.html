Modelos de Classificação e Métricas de Desempenho

  • GitHub

Motivação

Muitas vezes temos conjuntos de dados cujas observações pertencem a determinadas classes e queremos ser capazes de prever a classes de outras observações, esse é um problema de aprendizado supervisionado. Outras vezes, temos dados e queremos agrupar as observações de acordo com suas semelhanças, esse é um caso de aprendizagem não supervisionada.

Assim, a fim de ter uma referência para projetos futuros, apliquei métodos de Classificação e Agrupamento a diferentes conjuntos de dados.

Dados

  2. College dataset
  3. LendingClub dataset
  4. Whine Quality dataset
